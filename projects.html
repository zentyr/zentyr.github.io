  • Neovim

MyShell

I worked with a friend to create our own custom shell using the C programming language. This custom shell utilized multithreading to execute requested programs from the user. We also manually implemented redirection, wildcards, and piping as well as several core Linux commands such as cd, ls, and exit. We also did this using only POSIX commands.

Tools used:

  • C programming language
  • Multithreading
  • Vim

Movie Helper Discord Bot

I created a custom bot for a Discord server which allowed users to create LetterBoxd-styled movie lists unique to the server. All users were allowed to add movies to this list, and server admins could mark the films as watched. The “rating” was calculated based on the average rating of all users from the server.

Tools used:

  • Python
  • Discord.py
  • Pycharm
